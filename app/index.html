Программа обучения

Модуль 1 - Основы

6 дней

  • День 1: Переменные, типы данных и преобразование типов
  • День 2: Базовые и логические операторы, условные ветвления
  • День 3: Циклы и конструкция "switch"
  • День 4: Function Declartion
  • День 5: Function Expression
  • День 6: Стрелочные функции

Практика модуля:

  • Работа с условиями: уменьшение числа до нуля при определенных условиях
  • Работа с массивами: поиск максимальной суммы строки в двумерном массиве
  • Работа со строками: поиск предложений с максимальным количеством слов
